## 📌 نظرة عامة | Overview

تحسين تكلفة التوكن والموجهات يهدف إلى تقليل التكلفة المرتبطة باستخدام واجهات برمجة تطبيقات (APIs) النماذج اللغوية، من خلال تحسين طريقة بناء وإرسال الطلبات.

Token and Prompt Cost Optimization aims to reduce costs associated with LLM API usage by improving how requests are constructed and submitted.

## 🎯 لماذا تحسين تكلفة التوكن مهم؟

- **تكلفة API:** Prompt Caching يوفر 41-80% من تكلفة الـ API في تطبيقات RAG والوكلاء.
- **الكمون:** TTFT أقل بنسبة 13-31% عند استخدام التخزين المؤقت.
- **تأثير مباشر:** كل توكن مُرسل = تكلفة مباشرة — تحسين بناء الطلبات يوفر مالاً فوراً.
