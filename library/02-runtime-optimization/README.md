# 2. تحسين التشغيل | Runtime Optimization

> **المسار:** المكتبة ← تحسين التشغيل
> **الوصف:** كيف نشغّل النموذج بأقل تكلفة — التجميع، فك الترميز، إدارة الذاكرة، التخزين المؤقت، ومحركات الاستدلال.
> **الوفر النموذجي:** 40-80% تحسن في الإنتاجية (Mirantis 2026)

## الفئات الفرعية

| الفئة | الوصف | الإدخالات |
|-------|-------|----------|
| [التجميع](batching/) | التجميع المستمر والدفعي | 1 |
| [فك الترميز](decoding/) | فك الترميز التخميني | 1 |
| [ذاكرة KV](kv-cache/) | PagedAttention, FlashAttention, RadixAttention | 3 |

| [محركات الاستدلال](serving-engines/) | مقارنة المحركات | 0 (→ comparisons/) |

## القاعدة الذهبية
