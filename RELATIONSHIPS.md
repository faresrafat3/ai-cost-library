## 2. التركيبات الموصى بها حسب السيناريو

### 🏆 "الوصفة الذهبية" لخدمة استدلال رخيصة (2026)

```
الطبقة 1: FP8 Quantization → نصف الذاكرة + مضاعفة الإنتاجية
الطبقة 2: vLLM + Continuous Batching → 3-4× استغلال GPU
الطبقة 3: Model Routing → 60-85% تقليل لأنك لا تحتاج Opus لكل شيء
الطبقة 4: Prompt Caching → 50-90% تقليل للبادئات المتكررة
───────────────────────────────
المحصلة: 80-95% تقليل تكلفة مقابل الاستخدام الساذج
```

### 🤖 "وصفة الوكلاء الاقتصادية"

```
الطبقة 1: توجيه المهام الفرعية لنماذج رخيصة → 60-80%
الطبقة 2: AgentDiet (ضغط المسار) → 21-36%
الطبقة 3: SkillReducer (ضغط المهارات) → 27%
الطبقة 4: حدود إنفاق + مراقبة → حماية من الانفجار
───────────────────────────────
المحصلة: 70-90% تقليل مقابل وكيل بدون تحسين
```
