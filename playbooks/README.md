# أدلة تطبيقية | Playbooks

## 📖 ما هي الأدلة التطبيقية؟

كل دليل تطبيقي (Playbook) هو مرجع عملي مُصمّم لمساعدة المطورين والمهندسين على تطبيق تقنيات تقليل تكلفة الذكاء الاصطناعي خطوة بخطوة.

Each Playbook is a practical, step-by-step guide designed to help developers and engineers implement AI cost-reduction techniques in real-world scenarios.
