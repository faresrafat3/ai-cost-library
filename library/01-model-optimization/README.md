# 1. تحسين النموذج | Model Optimization

> **المسار:** المكتبة ← تحسين النموذج
> **الوصف:** كل ما يتعلق بجعل النموذج نفسه أرخص — تصغيره، تكميمه، اختيار الحجم المناسب، أو استخدام بنية أكثر كفاءة.
> **الوفر النموذجي:** 30-75% تقليل تكلفة (Mirantis 2026)

## الفئات الفرعية

| الفئة | الوصف | الإدخالات |
|-------|-------|----------|
| [التكميم](quantization/) | تقليل دقة الأوزان (FP8, INT8, INT4) | 5 |
| [الضغط](compression/) | استخلاص المعرفة، القصّ | 2 |
| [اختيار النموذج المناسب](right-sizing/) | توجيه النماذج، قوانين التحجيم | 2 |
| [البنية الفعّالة](efficient-architecture/) | MoE, MoD, الإنهاء المبكر | 2 |

## القاعدة الذهبية
> **ابدأ بالنموذج الأرخص الذي يحقق الحد الأدنى من الجودة، ثم صعِّد فقط عند الفشل.**
> "Default cheap, escalate on evidence." (Digital Applied 2026)
