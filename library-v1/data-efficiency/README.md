# كفاءة البيانات | Data Efficiency

![Category Status](https://img.shields.io/badge/Status-Seeded-yellow)

## 📌 نظرة عامة | Overview

كفاءة البيانات (Data Efficiency) تُعنى بتقليل كمية وجودة البيانات المطلوبة لتدريب وضبط النماذج، مما يقلل تكاليف جمع البيانات، تنقيحها، وتخزينها.

Data Efficiency focuses on reducing the quantity and improving the quality of data needed for training and fine-tuning models, thereby lowering data collection, curation, and storage costs.
