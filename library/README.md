## المصدر العلمي للتصنيف

- **Spheron Network (2026):** إطار الأربع طبقات (Model → Runtime → Infrastructure → FinOps)
- **TrueFoundry (2026):** أربعة أبعاد تكلفة (Inference + Infrastructure + Agent + Operational)
- **Mirantis (2026):** أربع طبقات تحسين مع أرقام وفر نموذجية
- **Gartner (2026):** تأثير الوكلاء (5-30× مضاعف التوكن)
- **Epoch AI (2026):** مفارقة الاستدلال (الأسعار تنخفض، الإنفاق يرتفع)
