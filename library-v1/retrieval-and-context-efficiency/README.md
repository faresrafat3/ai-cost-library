## 📌 نظرة عامة | Overview

كفاءة الاسترجاع والسياق (Retrieval and Context Efficiency) تُعنى بتحسين تكلفة وفعالية أنظمة RAG والاسترجاع المعزّز بالتوليد، مع تقليل عدد التوكنات المطلوبة لكل استعلام.

Retrieval and Context Efficiency covers optimizing the cost and effectiveness of RAG systems and retrieval-augmented generation, while reducing the number of tokens needed per query.
